# Command for windows

## Compile rust code to webassembly
```bat
cd wasm
npm run wasm
```

## Compile files for web site by webpack
```bat
cd web
npx webpack
```

## Start web server
```bat
cd web
npm run serve
```

## Initializing for development
```bat
init.bat
```

## Link webassembly files to web site
```bat
link.bat
```
